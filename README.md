# ASP_Timetabling

Em cần chạy 3 lần với dataset với độ lớn lần lượt là 600, 900, 1280 ạ

## Hướng dẫn chạy

1. Clone/download dự án về máy
2. Chạy chương trình:
   - Với dataset độ lớn 600
   ```bash
   ./run.sh 1
   ```
   - Với dataset độ lớn 900
   ```bash
   ./run.sh 2
   ```
   - Với dataset độ lớn 1280
   ```bash
   ./run.sh 3
   ```
3. Chạy xong kết quả sẽ lưu vào file filename_result.txt ạ
